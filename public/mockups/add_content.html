 Add Musical Album
Is a collection of tracks with a picture and description, it can be made downloadable: the system will make a zip file with mp3's of your tracks, track list, and a text file that contains a link to the music, you can collect contributions for the album. It is essential that you have the rights to distribute the music you make available for downloads and collect contributions for. Music album is going to appear in the upper right corner of your profile in download and contribute section.




 Add Track
Display it on your profile, add to favorites, can't download it, should be an mp3 of any quality, with ID3 tags, you should either own recording rights or owner should have given you promision to post it on Kroogi, and you can prove it. Other people can listen to it, vote on it, comment on it, contribute money to it (if you have payment system connected), and add it to their favorites. You can submit your track to someone else's inbox or a contest.




 Add Video
You need to upload your video on youtube first and get a code and then embed it into Kroogi. After that, the same rules that apply to Track, apply to video




 Add Picture
You can upload a picture to kroogi, post it on your profile, you can download the original photo (same size), all other rules are same as video and track




 Add Writing
You can write it as plain text or you can beautify them by adding images and formatting it using some basic html code (writing supports some html and kroogi-specific tags). Here's a list of html codes that Kroogi Writing supports: http://kroogi.kroogi.com/content/show/4538




 Add Folder
older is a collection of any content. It has two functions: 1. it may be a grouping of content 2. you can make a folder downloadable. If a folder is downloadable you can add a zip file to the folder which can be downloaded by other kroogi users. You can attach multiple zipfiles. Users can download any or all zipfiles. Content cannot be downloaded without zipfiles. Must have rights to distribute the content.




 Add Announcement
Same as writing but it goes into the announcements section of your profile. Cannot be downloaded.
